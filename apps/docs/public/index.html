📦 核心包

@linch-kit/core
基础设施 - 插件系统、配置管理、日志
@linch-kit/schema
Schema 引擎 - 代码生成、验证
@linch-kit/auth
认证权限 - NextAuth + CASL
@linch-kit/crud
CRUD 操作 - 类型安全、权限集成
@linch-kit/trpc
API 层 - 端到端类型安全
@linch-kit/ui
UI 组件库 - shadcn/ui + 企业组件
@linch-kit/console
管理平台 - 多租户、权限管理

安装: npx create-linch-kit@latest my-app
