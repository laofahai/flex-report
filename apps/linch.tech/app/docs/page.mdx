## Quick Start

Get started with Linch Kit in just a few minutes:

```bash
npx create-linch-app my-app
cd my-app
npm run dev
```

That's it! You now have a fully functional application with:
- Auto-generated admin interface
- Type-safe API endpoints
- Database integration
- Authentication system
