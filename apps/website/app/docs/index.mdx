# LinchKit 文档

欢迎来到 LinchKit 官方文档！LinchKit 是一个 AI-First 的全栈开发框架，专为现代企业级应用开发而设计。

## 🚀 快速开始

如果您是第一次使用 LinchKit，我们建议您从以下步骤开始：

1. [快速开始](/docs/getting-started) - 安装并创建您的第一个 LinchKit 应用
2. [核心概念](/docs/core-concepts) - 了解 LinchKit 的设计理念和核心特性
3. [开发指南](/docs/guides) - 通过实际示例学习如何构建应用

## 📦 核心特性

### Schema 驱动架构
以 Zod Schema 为单一数据源，自动生成类型、API 和 UI 组件，确保数据一致性。

### 端到端类型安全
从数据库到前端的完整 TypeScript 类型链，编译时发现错误，运行时零类型问题。

### AI 优先设计
架构和 API 设计优先考虑 AI 理解和处理，提升开发效率和代码质量。

### 企业级功能
内置多租户、RBAC 权限、审计日志、监控告警等企业级功能，开箱即用。

## 🏗️ 架构概览

LinchKit 采用分层架构设计：

```
L0: @linch-kit/core      - 基础设施 (日志、配置、插件)
L1: @linch-kit/schema    - Schema 引擎 (定义、验证、转换)
L2: @linch-kit/auth      - 认证权限 (认证、授权、会话)
L2: @linch-kit/crud      - CRUD 操作 (通用数据操作)
L3: @linch-kit/trpc      - API 层 (类型安全 API)
L3: @linch-kit/ui        - UI 组件 (现代组件库)
L4: applications         - 应用层 (业务应用)
```
